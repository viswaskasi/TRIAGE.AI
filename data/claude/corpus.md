# Claude Help Center
- **Billing / Payments**: For failed Pro subscription charges, please ask the user to update their payment method in the billing settings. We will automatically retry the charge within 3 days.
- **API Access**: If a user hits a rate limit, advise them to check their usage dashboard or upgrade their tier. We cannot manually increase limits for free accounts.
- **Bugs/Glitches**: If Claude generates unexpected characters, please ask the user to clear their browser cache or try a different browser.
