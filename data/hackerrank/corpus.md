# HackerRank Support
- **Login Issues**: If a user cannot log in due to an invalid password, advise them to use the "Forgot Password" link on the login page to reset their credentials.
- **Assessments**: If an assessment link has expired, advise the user that they must request a new link directly from the company that invited them; HackerRank cannot extend deadlines.
- **Fraud**: If there is suspicion of plagiarism or fraud during a test, we will investigate the session logs. We do not provide details of the investigation to the candidate.
